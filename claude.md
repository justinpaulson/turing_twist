# Turing Twist - Design System

## Design Theme

Turing Twist uses a **black and white newspaper style** combined with **old school Nintendo graphics** aesthetic.

### Visual Style Guide

**Color Palette:**
- Primary: Black (#000000)
- Secondary: White (#FFFFFF)
- Accent: Dark Gray (#333333)
- Light Gray: (#CCCCCC) for borders and subtle elements

**Typography:**
- Headings: Bold, newspaper-style fonts (system serif fonts)
- Body: Clean, readable monospace or pixel-style fonts reminiscent of early gaming systems
- Pixel art aesthetic for decorative elements

**Design Elements:**
- Sharp borders (no rounded corners unless retro-gaming context)
- Dotted/dashed lines to mimic newspaper print
- Heavy use of borders and boxes to section content
- Pixelated graphics and icons
- High contrast black/white design
- Halftone patterns for texture (optional)

**Button Style:**
- Chunky, pixel-style buttons
- Black borders with white backgrounds
- Hover states with inverted colors (white on black)
- No gradients or shadows (flat design)

**Form Elements:**
- Bold borders
- Monospace fonts for input
- Clear labels in bold
- Newspaper-style section headers

**Layout:**
- Grid-based layouts similar to newspaper columns
- Clear visual hierarchy
- Boxed sections with thick borders
- Vintage gaming console UI inspiration

### Implementation Notes

- Use CSS borders extensively
- Implement pixel-perfect spacing
- Keep animations minimal and sprite-based when used
- Maintain high readability despite retro aesthetic
- All interactive elements should have clear hover states
- Use ASCII art or pixel graphics for decorative elements where appropriate

### References

- 1980s Nintendo Game Boy aesthetic
- Classic newspaper front page layouts
- Early computer terminal interfaces
- Retro gaming UIs (NES, SNES era)
